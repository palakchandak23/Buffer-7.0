#FollowUpGhost - Smart Email Tracker System

Problem Statement:
Automated Email Commitment Tracking System-
Email communication frequently contains implicit commitments such as promises to send information, schedule meetings,or provide updates.
These commitments are not automatically tracked by existing email systems, making them difficult to manage and easy to forget.
There is a need for a system that can automatically detect commitment-related statements in sent emails, extract relevant task and deadline information, 
and organize them into a structured system that prioritizes, tracks, and identifies overdue commitments.
